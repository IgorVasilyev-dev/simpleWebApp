Тестовое задание [Java Mastery]:


У нас есть база сотрудников (employeedb). Нам необходимо добавлять новых сотрудников, удалять уволенных, обновлять данные о сотрудниках, а также должна быть возможность просмотреть весь список сотрудников сразу и отдельно получить каждого сотрудника.

Для этого требуется разработать веб-приложение (Main App), представляет собой REST сервис, которое должно позволять выполнять CRUD операции над сотрудниками.
